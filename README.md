A complete test automation project built with Playwright and TypeScript. It tests a full e-commerce website from login to checkout.

What does it test ?
- Login & Register: sign in or create a new account
- Home Page: browse recently added products
- Search: search for a product and get results
- Product Page: view product details and manage quantity
- Cart: add, update, and remove products
- Checkout: fill shipping info and place an order

Tech used ?
- Playwright
- TypeScript
- Page Object Model
- Fixtures
- Faker.js (generates realistic fake data)
- HTML Report
